Variables: Las variables son como pequeñas cajas, que en su interior pueden almacenar datos en particular, desde un conjunto de letras (strings), números o booleanos (verdadero o falso). Sirven para introducirles valores o para guardar algo que necesitemos.
Strings: Los string son todos los carácteres, puede ser uno solo (string), o pueden ser muchos juntos (strings). A estos últimos se les llama cadena de strings. Siempre para ser considerados strings, van encerrados entre comillas simples(''), o dobles("").
Funciones (argumentos, `return`):Cuándo escribimos código y este se torna repetitivo, lo mejor es crear funciones, para dejar de repetir tantas veces el mismo código. Estas plantean soluciones a problemas reiterados que se nos presentan. 
Los argumentos, son aquellos parametros que recibe una funcion y que son como una especia de variable integrada a la función. Estos argumentos son los que va a requerir la propia funcion para poder desarrollar la resolucion.
Return, cuando necesitamos que la función devuelva algún valor, utilizamos return. Ahí le estamos diciendo a la funcíon que tiene que retornar un valor que es lo que justamente necesitamos. 
Declaraciones `if`:Es la manera de resolver aquellos problemas que presentan que una condición se debe cumplir para poder avanzar. Por ejemplo: si una condicion se cumple o no le voy a pedir a la computadora que ejecute un determinado código, si esa condición no se cumple ejecutara otro pedazo de código
 Valores booleanos (`true`, `false`): Los booleanos sirven para identificar si es verdadero (true) o falso (falso).
